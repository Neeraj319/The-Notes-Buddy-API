Welcome To API Guide

Notes: Variables are denoted as ${x}
All routes are after notes.awebisam.com/
Example:notes.awebisam.com/api/v1

    Authentication- Register

    To register user, send POST request to /api/v1/account/register with 'username, email, first_name, last_name and password'.

  • Authentication- Login

    To login user, send POST request to /api/v1/account/login with 'username and password'. Then, it will return user's Token that should be passed as header while fetching, updating and/or deleting as in next two steps.

  • Fetching

    To fetch notes, send GET request to /api/v1 including 'Authorization' header with value: 'Token ${userToken}'.
    To fetch specific note, send GET request to /api/v1/${noteId} including 'Authorization' header with value: 'Token ${userToken}'.

  • Posting/Deleting

    To POST/DELETE notes, send POST/DELETE request to /api/v1/${noteId} including 'Authorization' header with value: 'Token ${userToken}'.
